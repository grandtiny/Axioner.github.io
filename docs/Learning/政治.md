# 政治

## 1 第一套

*   1.1 马克思主义
    *   马克思主义的本质属性是人民性
    *   马克思主义的政治立场是人民至上
    *   马克思主义首要的和基本的观点是实践观点
    *   马克思主义的基本特征包括：科学性、人民性、实践性、发展性
    *   马克思主义追求的根本价值目标（也是共产主义的根本特征）是实现人的自由而全面的发展
*   1.2 人工智能
    *   价值评价有科学和非科学之别，评价结果有正确和错误之分
    *   意识在内容上是客观的，在形式上是主观的
    *   科学的任务在于准确分辨真象和假象，透过现象把握本质
    *   无论在感性认识还是理性认识中都有非理性因素的作用
    *   人们对事物本质的认识是一个不断深化的过程
*   1.3 使用价值
    *   使用价值是价值的物质承担者，也是交换价值的物质承担者，价值寓于使用价值之中
    *   决定商品交换的比例的不是商品的使用价值而是价值
    *   劳动是创造价值的唯一源泉，劳动力的使用即劳动，所以劳动力商品的使用价值就是价值的源泉
*   1.4 资本主义
    *   推动资本积累方式从一般产业资本积累向国际金融垄断资本积累转变，国际金融资本的垄断成为当代资本主义最突出、最鲜明、最主要的特征
*   1.5 实事求是
    *   实事：客观存在的一切事务
    *   是：客观事物的内部联系，即规律性
*   1.6 三个代表
    *   十七大提出“中国特色社会主义理论体系”的科学概念，标志着中国特色社会主义理论和实践的进一步成熟
    *   十八大创立了习近平新时代中国特色社会主义思想，谱写了新时代中国特色社会主义新篇章
*   1.7 习近平新时代中国特色社会主义思想
    *   “十个明确”是思想的主题内容，构成了这一思想体系的四梁八柱，发挥着**统摄作用**
    *   “十四个坚持“构成了新时代坚持和发展中国特色社会主义的**基本方略**
    *   “十三个方面成就”**全景式地展**示了习近平新时代中国特色社会主义思想的理论与实践成果
    *   “六个必须坚持”彰显了习近平新时代中国特色社会主义思想的**理论品格和鲜明特征**

*   1.8 现代化
    *   现代化的最终目标是实现人自由而全面的发展
    *   共产主义社会三个基本特征：物质财富极大丰富、消费资料按需分配；社会关系高度和谐、人们精神境界极大提高；实现人的自由而全面发展

*   1.9 高质量发展
    *   加快实现高水平科技自立自强，是推动高质量发展的必由之路
    *   加快构建新发展格局，是推动高质量发展的战略基点
    *   推进农业现代化是实现高质量发展的必然要求
    *   人民幸福安康是推动高质量发展的最终目的
    *   更好统筹质的有效提升和量的合理增长是推动高质量发展的重要举措

*   1.10 新发展格局
    *   **牢牢守住安全发展这条底线是构建新发展格局的重要前提和保障**
    *   以新安全格局保障新发展格局，才能夯实我国经济发展的根基
    *   以新安全格局保障新发展格局，必须统筹维护国家安全各类要素、各个领域、各方资源、各种手段，主动塑造于我有利的外部安全环境

*   1.11 新文化运动
